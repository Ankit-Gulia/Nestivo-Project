# Nestivo

Nestivo is a full-stack property listing web app inspired by modern stay-booking platforms. Users can browse properties, view stay details, sign up or log in, and authenticated users can add and manage property listings.

## Features

- Property listing and detail pages
- User signup, login, logout, and protected routes
- Add and edit property workflows
- Review routes for property feedback
- Image upload support with Cloudinary
- MongoDB database integration with Mongoose
- React frontend with Vite and Tailwind CSS
- Express backend with cookies, JWT authentication, and validation

## Tech Stack

**Frontend**

- React
- Vite
- Tailwind CSS
- React Router
- Axios
- React Hot Toast

**Backend**

- Node.js
- Express
- MongoDB
- Mongoose
- Joi
- JWT
- Cloudinary
- Multer
- Cookie Parser
- CORS
